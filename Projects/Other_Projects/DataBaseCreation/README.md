# Database Creation Project

This was a group project completed as part of a Database Management class, in collaboration with four other classmates. The project involved creating and optimizing a database using **MySQL** based on a publicly available dataset from GitHub.

---

## Project Overview

- Designed and optimized database tables with proper **primary and foreign keys**  
- Implemented **views** to simplify complex queries  
- Ran **queries** to analyze and extract insights from the dataset  
- Optimized database performance through **indexing and normalization**

> **Note:** This project is currently not set up to be run directly. The SQL scripts and dataset are included for reference and can be used as a starting point for building or testing the database.
