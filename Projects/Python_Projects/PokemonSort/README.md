# Pokémon Sorter

A Python application that sorts Pokémon data based on different metrics such as height, weight, base experience, and stats. The project uses a **merge sort algorithm** for sorting and a local JSON dataset (`pokemonDB_dataset.json`) containing Pokémon information.

---

## Technologies

- **Python 3**  
- **JSON** – for reading the dataset  
- **pprint** – for clean console output  

---

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/FranklinW10/Franklins_Projects.git
cd Franklins_Projects/Projects/Python_Projects/PokemonSort
```
2. Ensure the JSON dataset (pokemonDB_dataset.json) is in the same folder as the script.

3. Run the Script:

```bash
python3 PokemonSort.py
```
4. Follow Prompts

## Features

- Sorts Pokémon using a **merge sort algorithm**  
- Sortable metrics include:
  - Height, Weight, Base Experience  
  - HP, Defense, Attack, Speed  
- Interactive prompts allow repeated sorting with different metrics  
- Outputs sorted Pokémon data in a clean, readable format
