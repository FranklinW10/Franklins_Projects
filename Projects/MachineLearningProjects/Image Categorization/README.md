# Image Categorization with Machine Learning

This project classifies images into categories using a custom **CNN** and pre-trained models (**ResNet, AlexNet, VGGNet**) for comparison. Useful in applications like medical imaging, content moderation, and security.

---

## How to Run

Run directly in **Google Colab**:

1. Open the notebook:  
   [Project2_CS167.ipynb](https://github.com/FranklinW10/Franklins_Projects/blob/main/Projects/MachineLearningProjects/Image%20Categorization/Project2_CS167.ipynb)  
   or via Colab:  
   [Open in Colab](https://colab.research.google.com/github/FranklinW10/Franklins_Projects/blob/main/Projects/MachineLearningProjects/Image%20Categorization/Project2_CS167.ipynb)

2. Set runtime to **Python 3**:  
   `Runtime` → `Change runtime type` → `Python 3`

3. Run cells sequentially (**Shift + Enter**).  

>  An internet connection is required for datasets.

## 🔍 Key Features

- Preprocessing: resize, normalize, split data  
- Models: custom CNN, ResNet, AlexNet, VGGNet  
- Training & Evaluation: accuracy, loss, confusion matrices  
- Visualization: sample predictions and metrics
