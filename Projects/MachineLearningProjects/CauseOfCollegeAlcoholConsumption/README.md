# Cause of College Alcohol Consumption

This project analyzes factors influencing alcohol consumption among college students. Using machine learning techniques, it identifies key predictors and explores their impact on student drinking behavior. 
More information on what is going on through the project is given in Google Colab.

---

## Technologies & Libraries

- **Python 3.x**
- **Jupyter Notebook / Google Colab**
- **pandas** – data manipulation
- **numpy** – numerical operations
- **matplotlib / seaborn** – data visualization
- **scikit-learn** – machine learning models

---
## How to Run

This project was created in **Google Colab**, so you can run it directly in the cloud without installing anything locally.

### Steps:

1. Open the notebook in Google Colab:
   [Franklin_Wohnoutka_Machine_Learning_Project1-2.ipynb](https://github.com/FranklinW10/Franklins_Projects/blob/main/Projects/MachineLearningProjects/CauseOfCollegeAlcoholConsumption/Franklin_Wohnoutka_Machine_Learning_Project1-2.ipynb)

2. Click the **“Open in Colab”** button in the top right corner (or open the notebook in Colab via this link: [https://colab.research.google.com/github/FranklinW10/Franklins_Projects/blob/main/Projects/MachineLearningProjects/CauseOfCollegeAlcoholConsumption/Franklin_Wohnoutka_Machine_Learning_Project1-2.ipynb](https://colab.research.google.com/github/FranklinW10/Franklins_Projects/blob/main/Projects/MachineLearningProjects/CauseOfCollegeAlcoholConsumption/Franklin_Wohnoutka_Machine_Learning_Project1-2.ipynb))

3. Make sure the notebook has **runtime type set to Python 3**:  
   - Go to `Runtime` → `Change runtime type` → Select `Python 3` → Save

4. Run the notebook cells sequentially by pressing **Shift + Enter** or clicking the **Play** button on each cell.

5. Visualizations and results will appear directly in Colab.

> Note: The notebook fetchs data from external sources, so an **active internet connection** is required.
