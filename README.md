## How to Run
Each project folder has its own setup instructions. In general:

- **Java Projects**
  - Compile with `javac` and run with `java`, or open in an IDE like IntelliJ or Eclipse.
- **Python Projects**
  - Create a virtual environment.
  - Install dependencies with:  
    ```bash
    pip install -r requirements.txt
    ```
  - Run scripts directly with `python file.py` or launch Jupyter notebooks.
